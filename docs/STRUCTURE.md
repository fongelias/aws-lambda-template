### Directory Descriptions
|name|description|
|----|-----------|

|docs|stores documentation|
|STRUCTURE.md|see above|
|CREATE_LAMBDA.md|instructions on how to create a lambda|
|src|stores code for lambdas|
|src/templates|code for lambda templates|
